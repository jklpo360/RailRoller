# RailRoller
Table-top assistant application for Rail Baron (1977)

### Functionality:
Rail Roller tracks each player's destinations, rewards, and home cities to make the experience of playing Rail Baron easier to teach and faster than ever!

### How to start:
1. Ensure that you have Python installed - it should work for any version of python3
2. Launch RailRoller.py by using the command ```py RailRoller.py``` or right-click RailRoller.py in File Explorer and select Open with -> Python
3. Enter the number of players and press enter to start the assistant
